# Ping Pong

#### 3rd Project - Epicodus, 06.25.2017

## Description

This is my third project in Epicodus- A "Ping Pong" detector where the rules as follows:
- Numbers evenly divisible by 3 become "ping"
- Numbers evenly divisible by 5 become "pong"
- Numbers evenly divisible by 15 become "pingpong"

## Setup/Installation Requirements

* Clone this Repository
* Open into web browser of your choice
* Enter numbers to see the script run based on the rules applied in the description
